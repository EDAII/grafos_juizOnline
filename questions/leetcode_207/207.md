## A questão

A questão consiste em determinar se é possível completar todas as
disciplinas de um curso, dadas as relações de pré-requisitos entre elas.

O objetivo é retornar **true** caso seja possível concluir todas as
disciplinas, ou **false** caso exista alguma dependência circular.

![207 Questao](../../assets/207/207_question.png)

## Estratégia

Para resolver o problema, as disciplinas foram modeladas como um grafo
direcionado:

-   Cada disciplina é representada como um **vértice**.
-   Cada pré-requisito gera uma **aresta dirigida** do curso obrigatório
    para o curso que depende dele (`b → a`).

A partir dessa modelagem, o problema vai para verificar se esse grafo
dirigido contém **ciclos**. Caso exista ciclo, significa que há
dependência circular entre disciplinas, o que torna impossível finalizar
o curso.

Para verificar a presença de ciclos, utilizamos um procedimento baseado
no **grau de entrada** dos vértices.
A ideia é que um vértice com grau de entrada zero representa uma
disciplina que pode ser cursada imediatamente, pois não depende de
nenhuma outra.

## Algoritmo utilizado - Processamento por grau de entrada (variação de BFS)

O método implementado segue os princípios apresentados no conteúdo:

1.  **Construção da lista de adjacência** do grafo.
2.  **Cálculo do grau de entrada** de cada disciplina.
3.  **Fila inicial** com vértices de grau zero.
4.  **Processo iterativo semelhante ao BFS**:
    -   Remover um vértice da fila (curso concluído).
    -   Reduzir o grau de entrada das disciplinas que dependem dele.
    -   Inserir na fila disciplinas que atingirem grau zero.
5.  **Verificação final**:
    -   Se todos os cursos forem processados, não há ciclo.
    -   Caso contrário, existe dependência circular.

Esse procedimento é uma forma de ordenação topológica baseada em grau de
entrada, compatível com o conteúdo estudado em aula.
